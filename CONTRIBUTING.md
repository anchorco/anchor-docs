# Contributing to Anchor Documentation

Thank you for contributing to Anchor's documentation!

## How to Contribute

1. **Fork the repository**
2. **Create a branch** for your changes
3. **Make your edits** to enhance Anchor documentation
4. **Submit a pull request** with a description of your changes

## Contribution License

By contributing to this repository, you agree that your contributions 
will be incorporated into Anchor's proprietary documentation. You grant 
Anchor a perpetual, worldwide, non-exclusive license to use, modify, 
and distribute your contributions as part of this documentation.

## What Can Be Contributed

- Documentation improvements (clarity, accuracy, examples)
- Bug fixes in code examples
- New examples or use cases
- Translation improvements
- Accessibility improvements

## Code of Conduct

- Be respectful and constructive
- Focus on improving the documentation
- Follow existing style and formatting

## Questions?

If you have questions about contributing, please open an issue or contact 
us at founders@getanchor.dev.
